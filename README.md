# 📊 Telecom X – Predicción de Cancelación de Clientes (Churn)

## 📌 Descripción del Proyecto

Este proyecto forma parte del desafío **Telecom X - Parte 2** del programa de **Data Science de Alura + Oracle**.

Después de realizar el **análisis exploratorio de datos (EDA)** en la primera etapa del proyecto, el objetivo ahora es desarrollar **modelos de Machine Learning capaces de predecir la cancelación de clientes (Churn)**.

La empresa **Telecom X** enfrenta una alta tasa de cancelación de clientes, por lo que busca implementar soluciones basadas en **inteligencia predictiva** para anticiparse a este problema y tomar decisiones estratégicas para mejorar la retención.

---

## 🎯 Objetivos del Proyecto

En este desafío se desarrollará un pipeline básico de Machine Learning que permita:

- Preparar los datos para el modelado.
- Realizar análisis de correlación y selección de variables.
- Entrenar modelos de clasificación para predecir la cancelación de clientes.
- Evaluar el rendimiento de los modelos utilizando métricas de evaluación.
- Interpretar los resultados obtenidos.
- Identificar los factores más influyentes en la cancelación de clientes.

---

## 🧠 Problema de Negocio

La cancelación de clientes representa una pérdida directa para la empresa. Identificar **qué clientes tienen mayor probabilidad de cancelar** permite a la compañía:

- Diseñar estrategias de retención.
- Ofrecer incentivos personalizados.
- Mejorar la experiencia del cliente.
- Reducir pérdidas financieras.

Este proyecto busca apoyar esas decisiones mediante **modelos predictivos de churn**.
